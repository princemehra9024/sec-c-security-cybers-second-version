## 🔧 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/princemehra9024/sec-c-security-cybers-second-version.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Configure Environment**:
   Create a `.env` file and add your MongoDB URI and JWT secrets.
4. **Run the server**:
   ```bash
   npm run dev
   ```
